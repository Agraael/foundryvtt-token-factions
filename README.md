# Tokens Faction (Lasossis's Fork)

Personal fork of [p4535992's Token Factions](https://github.com/p4535992/foundryvtt-token-factions). For everything not listed here, read the [original module's README](https://github.com/p4535992/foundryvtt-token-factions).

## Advanced Teams

Define your own teams with a name, a colour, and a default disposition. Assign tokens from the Token HUD. Right-click an Actor folder to assign every actor inside it.

![Team menu](./doc/advanced%20team%20menu.png)

The Disposition Matrix sets per-pair relationships between teams.

![Team matrix](./doc/advanced%20team%20exemple%20matrix.png)

Click a cell to cycle Friendly, Neutral, Hostile, Secret. Right-click reverses. Click a team name to set its row or column. The Symmetric view toggle hides the lower half and mirrors edits.

Lancer Automations and my forks of Grid-Aware Auras, Terrain Height Tools, and Template Macro read team dispositions from here.

## Keep border under sprite on hover

Module setting. When on, hovering a token leaves the faction border behind the artwork instead of lifting it above.
